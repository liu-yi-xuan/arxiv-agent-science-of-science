# 📚 AI Agents x Science of Science

[![Daily Paper Update](https://github.com/liu-yi-xuan/arxiv-agent-science-of-science/actions/workflows/daily_update.yml/badge.svg)](https://github.com/liu-yi-xuan/arxiv-agent-science-of-science/actions/workflows/daily_update.yml)

> **Why these two fields belong together:** AI agents are transforming how science is done — automating literature review, generating hypotheses, running experiments, and writing papers. At the same time, Science of Science provides the quantitative lens to understand how the agent research revolution itself unfolds — tracking knowledge diffusion, measuring impact, and mapping collaboration. They form a feedback loop: agents accelerate science, and science of science helps us build better agents.

This repo auto-updates daily from ArXiv, tracking papers at the intersection of AI agents and the scientific research process.

## 🏗️ Project Structure

```
├── README.md              # This file — auto-updated daily
├── fetch_papers.py        # ArXiv paper fetcher + classifier + TLDR generator
├── papers/                # Archive of daily paper fetches (JSON)
├── .github/workflows/     # GitHub Actions for daily automation
└── requirements.txt       # Python dependencies
```

## 🚀 How It Works

1. **Daily at 08:00 UTC**, a GitHub Action triggers `fetch_papers.py`
2. The script queries ArXiv for recent papers at the intersection of AI agents and science of science
3. Papers are classified into categories and a TLDR is generated
4. The README is auto-updated with the day's findings
5. Changes are committed and pushed automatically
